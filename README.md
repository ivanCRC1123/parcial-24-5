# Sistema de Gestión de Productos

## Descripción

Este proyecto es una aplicación fullstack desarrollada como trabajo práctico para la materia.

Permite gestionar productos, categorías e ingredientes mediante operaciones CRUD completas.

El sistema está compuesto por un backend en FastAPI con base de datos PostgreSQL y un frontend en React con TypeScript y Tailwind CSS.

---

## Tecnologías utilizadas

### Backend

- FastAPI
- SQLModel
- PostgreSQL
- Pydantic

### Frontend

- React
- TypeScript
- Tailwind CSS
- TanStack Query
- React Router DOM

---

## Funcionalidades

- Crear, editar, eliminar y listar productos
- Gestión de categorías e ingredientes
- Relación muchos a muchos entre productos e ingredientes
- Validaciones en backend y frontend
- Consumo de API con sincronización automática
